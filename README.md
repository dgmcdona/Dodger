This dodger style, space themed game was developed in the University of New Orleans Game Development club. Written in JavaScript and using the Phaser3 framework, it was developed under the guidance of Ted Holmberg. You can check out some of his other work at [his GitHub account](https://github.com/scalemailted). 

To run this game in your browser, visit [https://dgmcdona.github.io/Dodger/](https://dgmcdona.github.io/Dodger/)
